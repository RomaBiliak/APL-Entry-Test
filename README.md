### Task 1

|Mon |Tue| Wed| Thu| Fri| Sat| Sun|
| ---- |:----:|:----:|:----:|:----:|:----:| ----:|
|78°F | 80°F |89°F |82°F |79°F| 89°F| 73°F|

b. Calculate average temperature.
c. Find day(s) with highest temperature (First day of the week is Monday).
